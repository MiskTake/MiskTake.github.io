# 📄 CV en HTML

Este proyecto consiste en la creación de un **Currículum Vitae (CV)** utilizando únicamente HTML. Fue desarrollado como ejercicio práctico para aprender la estructura y las etiquetas fundamentales del lenguaje de marcado HTML.

---

## 🛠 Tecnologías utilizadas

- HTML5

---

## 🏷 Etiquetas HTML utilizadas

| Etiqueta | Descripción |
|---|---|
| `<header>` | Encabezado principal de la página |
| `<img>` | Inserción de imagen (foto de perfil) |
| `<h1>` | Título principal |
| `<section>` | Separación y organización de secciones |
| `<h2>` | Encabezados de sección |
| `<p>` | Párrafos de texto |
| `<strong>` | Texto en negritas |
| `<a>` | Enlaces / URLs |
| `<ul>` | Creación de listas desordenadas |
| `<li>` | Ítems de una lista |
| `<article>` | Artículos o bloques de contenido independiente |
| `<h3>` | Encabezados de tercer nivel |
| `<table>` | Creación de tabla |
| `<table border>` | Tabla con bordes visibles |
| `<thead>` | Agrupación del encabezado de la tabla |
| `<tr>` | Define filas dentro de la tabla |
| `<th>` | Encabezado de una columna o fila |
| `<tbody>` | Agrupa el cuerpo de filas de la tabla |
| `<td>` | Celda de datos dentro de la tabla |

## 🚀 ¿Cómo visualizarlo?

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador web preferido.
3. ¡Listo! No requiere ninguna instalación adicional.

---

## 📚 Aprendizajes

Con este proyecto se practicaron los conceptos básicos de HTML, incluyendo:

- Estructuración semántica de una página web.
- Uso de etiquetas de texto, listas y tablas.
- Inserción de imágenes y enlaces.
- Organización del contenido en secciones y artículos.
